# Strategy Presenter

Crea presentazioni strategiche professionali in stile slide-deck.

## Setup

```bash
npm install
npm run dev
```

## Deploy su Vercel

1. Push su GitHub
2. Importa su [vercel.com](https://vercel.com)
3. Deploy automatico

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- localStorage (persistenza browser)
- Lucide React (icone)
